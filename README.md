# 前言

React的项目模拟

## 项目名称

富途牛牛APP

## 技术栈

React+create-react-app+React-router+React-redux+antd+@antv/f2+scss+webpack+axios+ES6/7

## 项目运行

### 注意：由于涉及大量的ES6/7等新属性，node需要6.0以上的版本

```
    git clone https://github.com/gzh51904/Futuniuniu/

    cd ftnn

    npm install 或yarn

    npm start
```
